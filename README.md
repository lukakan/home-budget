JavaStart 24.1-JDBC-home work</br>
simple app to manage home budget - spring &amp; jdbc<br>

*Task details* <br>
Application must be made according to DAO pattern and provide following options
* add new transactions
* update transaction
* delete transaction
* show all incomes
